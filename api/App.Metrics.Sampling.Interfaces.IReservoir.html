Interface IReservoir

Provides access to a Resevoir Sampling implementation. Reservoir sampling is a family of randomized algorithms for randomly choosing a sample of k items from a list S containing n items, where n is either a very large or unknown number. Typically n is large enough that the list doesn't fit into main memory.

  • UniformReservoir
  • ExponentiallyDecayingReservoir
  • SlidingWindowReservoir.
  • HdrHistogramReservoir

Remarks

To specify the type of reservoir sampling to use either set the Default type globally using theAppMetricsOptions on startup, or use the

HistogramOptions or TimerOptions when defining these metrics types which support resevoir sampling. If no sampling type is specified the

ExponentiallyDecayingReservoir will be used.
